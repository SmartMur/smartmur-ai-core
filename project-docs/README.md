# Project Docs

This directory is the operational record for unattended work.

## Files

- `session-2026-03-01-night-ops.md` — current session log and actions completed.
- `work-queue.md` — prioritized next actions to continue execution without waiting.
- `telegram-reporting.md` — Telegram bot update behavior, setup, and flush instructions.

## Rules

1. Every material change gets logged here.
2. Each log entry includes what changed, why, and how to verify.
3. If automation/reporting changes are made, include rollback notes.
